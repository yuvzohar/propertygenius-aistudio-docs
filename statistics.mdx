The Statistics page provides performance analytics and insights, helping you compare your portfolio with the average investor.<br />

## Statistics Dashboard

The **Statistics** dashboard provides a high-level overview of your portfolio's performance. <br />Filter the data by relevant segments to gain actionable insights and monitor key metrics, including:

- **Income, Revenue & Expenses** - Analyze your properties' financial performance over time.
- **Tenant Performance** - Track tenant activity, rental income, and payment trends.
- **Occupancy Rate** - Monitor occupancy levels and rental performance over time.
- **Ready Phase Insights** - Discover AI-generated insights for properties in the **Ready** phase to help optimize your investment decisions.<br />

<Frame caption="Statistics Dashboard">
  ![Statistics Dashboard](/images/statistics-dashboard.jpg)
</Frame>

<Note>
  **Currency Display**

  - If your properties use **different currencies**, and the **All Properties** filter is selected, all statistics are automatically converted and displayed in **USD (\$)**.
  - If **all properties use the same currency** (for example, **EUR**), all statistics will be displayed in that currency instead of USD.
  - When viewing a **specific property**, all statistics are displayed in the currency assigned to that property.
</Note>
